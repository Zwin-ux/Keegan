# Roadmap

This roadmap aligns with the current codebase and expands toward the full radio platform.

## Phase 0 (done)
- Local C++ audio engine with moods, stems, and heuristics.
- Web UI prototype with tuner UX.
- Local HTTP server with /api/state.

## Phase 1 (next)
- Mod pack format and loader.
- Local bridge API expansion.
- Pack validator tool.
- Replace hardcoded mood lists in UI with data-driven lists.

## Phase 2
- Regional station registry service.
- Station pages, schedule, and chat.
- WebRTC ingest and LL-HLS playback.
- Basic mobile listener app (PWA or native shell).

## Phase 3
- Social features: follows, raids, co-listen rooms.
- Creator analytics and reputation system.
- Station discovery and regional charts.

## Phase 4
- Marketplace for mods and packs.
- Monetization options for creators.
- Federation support for self-hosted relays.
